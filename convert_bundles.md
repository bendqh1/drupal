## Installation

```shell
composer require 'drupal/convert_bundles:^2.0@beta'
```

## Patching (if needed)

If there is need of patching, then patch and proceed.

## Enabling

```shell
drush en convert_bundles
```

## Uninstallation

```shell
drush un convert_bundles
composer remove 'drupal/convert_bundles:^2.0@beta'
```
